# Prolog Family Tree Assignment

## Description
This assignment demonstrates the use of Prolog programming using SWI-Prolog.

## Tasks Done

- Installed and configured SWI-Prolog
- Ran basic Prolog programs
- Created a family tree

## Family Tree Features

The program includes:

- Grandparents
- Parents
- Children
- Grandchildren
- Cousins
- Uncles
- Aunts

## File Included

- family_tree.pl

## Sample Queries

```prolog
grandparent(john, david).
child(david, mike).
cousin(chris, david).
```
